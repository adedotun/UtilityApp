How to sign:

  1. Draw, add text, or insert date using tools on the left
  2. Click on the PDF preview to place items
  3. Drag items to reposition, click √ó to remove
  4. Navigate through pages if needed
  5. Click "Save Signed PDF" when done

How to edit:

  1. Select a tool (Highlight, Redact, Draw, Shapes, etc.)
  2. Adjust color, opacity, and size as needed
  3. Draw directly on PDF or click to place elements
  4. Use eraser to remove edits
  5. Navigate pages and edit multiple pages
  6. Click "Save Edited PDF" when done

Word to PDF conversion:

  1. Add one or more Word documents (.docx files)
  2. Documents will be converted in order
  3. Each document becomes part of the PDF
  4. Click "Convert to PDF" to download

Note: Formatting may vary slightly from original.

View Word documents:
